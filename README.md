# Project Pomo-mon 🍅👾

A gamified productivity application built with **Vanilla JavaScript** and **Firebase v10**. 

## Features
- **Pomodoro Timer:** Focused work sessions.
- **Mon Collection:** Unlock and grow creatures through focus.
- **Firebase Auth:** Secure Google Login for progress saving.
- **Mobile Optimized:** Built and hosted entirely via mobile.

## Tech Stack
- **Frontend:** HTML5, CSS3, Vanilla JS
- **Backend:** Firebase (Authentication, Firestore)
- **Hosting:** GitHub Pages

## Local Development
Since this is a single-file project, simply open `index.html` in any modern browser. Ensure your domain is whitelisted in the Firebase Console to enable authentication.
